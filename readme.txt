Allow the upload of pages, numbers and keynote files to WordPress

== Description ==

Simple plugin enabling the upload of pages, numbers and keynote files to WordPress.

It simply adds these mine types to the existing allowed mine types.  You will need to add the required file extensions to those allowed on the settings page.

Has been tested on both single and multisite WordPress installs.

== Installation ==

Standard plugin installation.  Upload the files to your server using the built in plugin installer or directly to the servers plugin folder.

Activate the plugin.

On older builds of WordPress you may need to add the .pages .numbers and .key files extensions to the allowed list on the settings page.
